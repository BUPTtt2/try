## 📱 APP介绍

**Tilo** 是一款文艺风格的生活记录APP，采用Jetpack Compose开发，集成了Supabase云端存储和ModelScope AI服务。

### 核心功能

- ✨ AI关键词生成文艺内容
- 📸 多图云端上传
- 🏷️ 标签分类筛选
- 🌸 梦幻粉紫配色

### 技术栈

| 技术 | 说明 |
|------|------|
| Kotlin | 开发语言 |
| Jetpack Compose | UI框架 |
| Material Design 3 | 设计语言 |
| Supabase | 云端数据库+存储 |
| ModelScope | AI内容生成 |
